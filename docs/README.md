# Portfolio Docs

이 폴더는 포트폴리오에 들어갈 텍스트와 이미지 원본을 관리하는 공간입니다.

## 구성

- `portfolio-content.md`: 페이지별 문구, 연락처, 연혁, 키워드
- `assets.md`: 이미지/시각 자료 관리 규칙과 현재 사용 중인 이미지 목록
- `images/README.md`: 앞으로 추가할 원본 이미지 보관 위치

## 관리 원칙

- 화면에 들어갈 문구를 바꾸기 전에 먼저 `portfolio-content.md`를 수정합니다.
- 코드에 바로 문구를 넣기보다, 이 문서를 기준으로 화면에 반영합니다.
- 웹에서 실제로 사용하는 최적화 이미지는 `assets/`에 두고, 원본/참고 이미지는 `docs/images/`에 둡니다.
- 공개 웹에 올리기 부담스러운 개인정보는 이 문서에도 최소한만 기록합니다.
